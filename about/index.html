About


I try not to blog unless what I am trying to convey could be useful and is not googleable. It is usually about tech and engineering. It is not meant to solve everyone’s problems, just something I find interesting and/or fun.

English is my third language, and this shows. Please let me know if you find any such issues.

My current contacts can be found at nagimov.com (javascript required).

Javascript and user-tracking

Pages on this site include the following javascript libraries:

  • remark commenting system which doesn’t spy on users (https://remark42.com)
  • fathom privacy-focused analytics system (https://github.com/usefathom/fathom)

Feel free to disable javascript entirely, the content is accessible as plain html.
